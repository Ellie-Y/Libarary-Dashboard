### How to run the application

1. Download and install all requirements for the server 

   ```
   npm install
   ```

2. Running the library serve

   ```
   node server.js
   ```
   
3. Install all requirements for the application

   ```
   npm install
   ```

4. Run the serve

   ```
   npm run serve
   ```
